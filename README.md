Simple dwmbar written in python

In WIDGET_ORDER you can specify the order in which widgets will be located. For now there are "time", "cpu", "memory", "battery", "volume".

> [!NOTE]
> 
> WIDGET_ORDER = ["time", "cpu", "memory", "battery", "volume"]

To use as a package on archlinux you can build a python file via nuitka. Then get sha256 and write it to PKGBUILD and build it.

Простой dwmbar написанный на python

В WIDGET_ORDER вы можете указать в каком порядке будут располагаться виджеты. Пока есть "time", "cpu", "memory", "battery", "volume".

> [!NOTE]
> 
> WIDGET_ORDER = ["time", "cpu", "memory", "battery", "volume"]

Чтоб использовать как пакет на archlinux вы можете собрать python файл через nuitka. После чего получить sha256 записать его в PKGBUILD и собрать.
